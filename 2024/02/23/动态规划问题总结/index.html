1.最长递增子序列

LC300 最长递增子序列

  • dp数组含义：dp[i]表示以nums[i]结尾的最长递增子序列的长度
  • dp数组初始化：dp[i] = 1
  • 状态转移方程：dp[i] = max(dp[j]) + 1, 0 <= j < i且nums[i] > nums[j]
  • 返回结果：max(dp[i])

LC673 最长递增子序列个数

  • dp数组含义：dp[i][0]表示以nums[i]结尾的最长递增子序列的长度，dp[i][1]表示以nums[i]结尾的最长递增子序列的个数
  • dp数组初始化：dp[i][0] = 1, dp[i][1] = 1
  • 状态转移方程：dp[i][0] = max(dp[j][0] + 1, dp[i][0]), 0 <= j < i且nums[i] > nums[j],对应dp[i][1] = dp[j][1]或dp[i][1] += dp[j][1]
  • 返回结果：遍历统计max(dp[i][0])的dp[i][1]

LC334 递增三元组

  • 动归解：从左向右遍历维护以当前位置为结尾的min数组，从右向左遍历维护以当前位置为结尾的max数组，遍历数据比较是否存在nums[i]>min[i]&&nums[i]<max[i]
  • 贪心解
